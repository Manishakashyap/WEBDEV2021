Pointers for block

  1. Width of a block level elment is 100% of the viewport
  2. Height of a block is not aet by default
  3. Height & width can be explicitly set using CSS

Pointers for inline

  1. Width of a inline level element is dependent on the content
  2. Height of a block is not aset by default
  3. Height & width can not be explicitly set using CSS
